# Personal Homework Management System

一个由学生开发的作业管理系统，帮助跟踪作业截止日期，减少逾期风险。
项目地址：https://github.com/952313/Personal-Homework-Management-System/

## ✨ 功能特点

- **作业记录**：轻松添加作业信息及提交日期
- **智能提醒**：颜色编码系统直观显示作业状态
  - 🔴 今天到期 - 红色
  - 🟡 3天内到期 - 黄色  
  - ⚪ 未过期（进行中，超过3天） - 正常颜色
  - ⚫ 已过期 - 粉色
- **风险管理**：降低作业迟交被扣分的概率
